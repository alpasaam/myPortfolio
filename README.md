# Saam Naeini - Portfolio Website

Personal portfolio website showcasing my work, experience, and projects as a Full Stack Software Developer.

## About

This is my personal portfolio website built with React, featuring my professional experience, projects, education, and achievements. The site highlights my work at BYU, my role as co-founder of GRAID, and various technical projects I've built.

## Live Site

Visit [saamn.dev](https://saamn.dev) to see the live portfolio.

## Technologies Used

- React
- SCSS/SASS
- Lottie Animations
- React Reveal
- Font Awesome Icons

## Local Development

To run this project locally:

```bash
# Install dependencies
npm install

# Start development server
npm start
```

The site will be available at `http://localhost:3000`

## Building for Production

```bash
# Create production build
npm run build
```

## Deployment

The site is deployed using GitHub Pages. To deploy:

```bash
npm run deploy
```
